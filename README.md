### Development and use
If you want to local install our app, you need make following commands in terminal:

```sh
$ git clone https://github.com/pmaxp/webportfolio.git
$ cd webportfolio
$ npm i
$ bower i
$ cd php
$ php composer.phar update
$ cd ..
$ gulp
```
